vltpack:IT-upload-tests

Full name:

net.adamcin:vltpack-maven-plugin:0.9.0-SNAPSHOT:IT-upload-tests

Description:

Upload integration test dependencies, including vault packages and OSGi bundles, to the configured integration test server

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.0.0.
  • Binds by default to the lifecycle phase: pre-integration-test.
