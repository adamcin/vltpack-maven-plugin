vltpack:IT-upload

Full name:

net.adamcin:vltpack-maven-plugin:1.1.0-SNAPSHOT:IT-upload

Description:

Uploads the project vault package and its dependencies to the configured IT server

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.0.0.
  • Binds by default to the lifecycle phase: integration-test.
